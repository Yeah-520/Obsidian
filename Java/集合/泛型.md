>定义一种模板
>泛型**不自动向上转型**
## 标记符
|      标记符      | 含义与常见用途                                                     |
| :-----------: | :---------------------------------------------------------- |
|    **`E`**    | **Element** - 主要用于**集合**框架，如 `ArrayList<E>`, `Iterator<E>`  |
|    **`T`**    | **Type** - 表示**任意类型**，最通用的标记符，用于类、方法、接口                     |
|    **`K`**    | **Key** - 用于**键值对**集合，如 `Map<K, V>` 中的 Key                  |
|    **`V`**    | **Value** - 用于**键值对**集合，如 `Map<K, V>` 中的 Value              |
|    **`N`**    | **Number** - 用于**数值类型**，如 `Number` 及其子类 (Integer, Double 等) |
| **`U` / `V`** | 表示**第二、第三个**任意类型 (当需要多个泛型参数时)                               |
|    **`?`**    | **Wildcard (通配符)** - 表示**未知类型**                             |
- 无限定通配符`<?>`很少使用，可以用`<T>`替换，同时它是所有`<T>`类型的父类
## 局限
1. 不能是基本类型，例如：`int`
2. 不能获取带泛型类型的`Class`，例如：`Pair<String>.class`
3. 不能判断带泛型类型的类型，例如：`x instanceof Pair<String>`
4. 不能实例化`T`类型，例如：`new T()`
## 上界通配符（? extends T）
>`? extends T` = `?` 只能是 T 或 T 的**子类**，**只读**安全，**不能写**（除了 null）
>适合：遍历、读取、统计、拷贝
```java
public class Main {
    public static void main(String[] args) {
        Pair<Integer> p = new Pair<>(123, 456);
        int n = add(p);
        System.out.println(n);
    }
	
    static int add(Pair<? extends Number> p) {
        Number first = p.getFirst();
        Number last = p.getLast();
        return first.intValue() + last.intValue();
    }
}

class Pair<T> {  
    private T first;  
    private T last;  
  
    public Pair(T first, T last) {  
        this.first = first;  
        this.last = last;  
    }  
  
    public T getFirst() {  
        return first;  
    }  
    public T getLast() {  
        return last;  
    }  
    public void setFirst(T first) {  
        this.first = first;  
    }  
    public void setLast(T last) {  
        this.last = last;  
    }  
}
```
## 下界通配符（? super T）
>`? super T` = 只能是 T 或 T 的**父类**，**只写**安全，**不能读**（除了 Object）
>适合：写入
```java
public class SuperTest {  
    public static void main(String[] args) {  
        Pair<Number> p1 = new Pair<>(12.3, 4.56);  
        Pair<Integer> p2 = new Pair<>(123, 456);  
        setSame(p1, 100);  
        setSame(p2, 200);  
        System.out.println(p1.getFirst() + ", " + p1.getLast());  
        System.out.println(p2.getFirst() + ", " + p2.getLast());  
    }  
  
    static void setSame(Pair<? super Integer> p, Integer n) {  
        p.setFirst(n);  
        p.setLast(n);  
    }  
}
```
## PECS原则
>Producer Extends Consumer Super
- 如果需要返回`T`，它是生产者（Producer），要使用`extends`通配符；如果需要写入`T`，它是消费者（Consumer），要使用`super`通配符
```java
public class Collections {
	public static <T> void copy(List<? super T> dest, List<? extends T> src) {  
	    for (int i = 0; i < src.size(); i++) {  
	        T t = src.get(i); // src是producer  
	        dest.add(t); // dest是consumer  
	    }  
	}  
}
```
