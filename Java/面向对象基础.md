# 术语
- 方法签名 (方法参数 + 方法名称)
- 参数列表 (多个参数之间用 ( , ) 连接)
- 成员变量包含(实例变量 + 静态变量(static))
# 核心关键字
- `this`（当前对象）
- `static`（静态变量 / 方法、静态代码块）
- `final`（常量、不可继承类、不可重写方法）
# 访问修饰符
|     修饰符     | 类内部 | 同一包 | 子类  | 其他包 |
| :---------: | :-: | :-: | :-: | :-: |
|  `public`   | ✔️  | ✔️  | ✔️  | ✔️  |
| `protected` | ✔️  | ✔️  | ✔️  |  ❌  |
|   默认（无修饰）   | ✔️  | ✔️  |  ❌  |  ❌  |
|  `private`  | ✔️  |  ❌  |  ❌  |  ❌  |
# 静态变量(static field)
- `className.field` **通过类名调用**
- 静态变量只有一个共享**内存空间**
	 共享数据
	 一改全变
- 无论修改哪个实例的静态变量，效果都是一样的
# 静态方法(static method)
- `className.method`**通过类名调用**
- 无需创建对象
# Java规定
- 包含 `public` 类的 `.java` 文件，**文件名**必须与该 **public 类名** 完全一致
- 每个 public 类都被设计成该文件的主要接口或**入口**点
# 类(Class)
```java
class Person {
	String name; //字段(Field)
	public Person() { } //[无参]构造方法
	private void sleep() { } //方法(Method)
}
```
- 用于实现**封装**、**继承**、**多态**、**抽象**
- 实现代码的复用、扩展、解耦，

## 字段(Field)
>(成员)变量
>\[访问修饰符] \[static] \[final] 数据类型 字段名 \[= 初始值];
```java
// [访问修饰符] 数据类型 变量名;
	private String name;
```
- ### 要点
	- 成员变量**包含**(实例变量 + 静态变量(static))
	- 成员变量(一般是私有`private`/ `protected`)
	- 实例变量 = 非静态成员变量
	- 只有在对象实例化`new Person()`后，字段才会获取到内存，这也正是它被称作“实例”变量的原因。
## 方法(Method)
```java
class Person {
	private String name;
	/*
	[访问权限] 返回类型 方法名(参数列表) {
		  方法体
	}
	*/
	public String setName(String name) { // 设置名字的方法
		this.name = name;
	}
	public String getName() { // 获得名字的方法
		return this.name;
	}  
}
```
- ### 参数列表
	- 基本数据类型
	- 引用数据类型（类、接口、数组）
- ### 作用
	- 将类的**字段(Field)私有化**(用`private`修饰)，不让外部直接访问
	- 提供**公共方法**(用`public`修饰) ，作为外部访问的 “唯一入口”
	- 目的是保护数据的**安全性和完整性**，同时隐藏类的内部实现细节
- ### this
	- 在类的**非静态方法**和**构造方法**中使用
	- 访问**当前**类的成员变量(本类内部)
	- 用于**区分**同名的**成员变量**和**局部变量**(避免局部变量名冲突)
	- 返回当前对象(return this)
## 构造方法(Constructor)
```java
class Person {
	private String name;
	//有参构造方法
	public Person(String name) { // 初始化成员变量
		this.name = name;
	}
}
```
- ### 要点
	- **初始化**成员变量
	- **没有返回类型**~~void~~
	- 必须与类**同名**
	- 只能加**访问修饰符**
	- 可以无参
