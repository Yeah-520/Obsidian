# 多态
>多态是同一个行为具有多个不同表现形式或形态的能力。
>>即一个父类可能有若干子类，各子类实现父类方法有多种多样，调用父类方法时，父类引用变量指向不同子类实例而执行不同方法，这就是所谓父类方法是多态的。
## 体现多态
1. **方法重载**
2. **方法重写**
	-  子类**重写**`@Override`父类的方法
3. **构造方法**
4. 子类**继承**父类
5. 父类引用指向子类的对象（**向上转型**）
## 方法重载
>这种**方法名相同**，但各自的**参数不同**，称为方法重载
```java
public class Overload {  
    public static void main(String[] args) {  
        Adder math = new Adder();  
        System.out.println(math.sum(1, 2)); // 使用有两个参数的sum  
        System.out.println(math.sum(1, 2, 3));  // 使用有三个参数的sum  
    }  
}  
  
class Adder {  
	
    public int sum(int a, int b) {  
        return a + b;  
    }  
	
    public int sum(int a, int b, int c) {  
        return a + b + c;  
    }  
}
```
## 重写
>*建议使用 `@Override` 注解*
>重写的方法必须和父类中的方法有着相同的名字
>重写的方法必须和父类中的方法有着相同的参数
>必须是 is-a 的关系（继承关系）
```java
public class Override {  
    public static void main(String[] args) {  
        Bike bike = new Bike();  
        bike.run(); // 自行车在跑  
    }  
}  
  
class Vehicle {  
    void run() {  
        System.out.println("车辆在跑");  
    }  
}  
  
class Bike extends Vehicle {  
    @Override
    void run() {  
        System.out.println("自行车在跑");  
    }  
}
```
### 规则
1. **只能重写继承过来的方法**
2. `final`、`static` 的方法**不能重写**
3. 必须是**相同**的**参数列表**和**返回类型**
4. 重写的方法不能使用限制等级更严格的权限修饰符
	- 如果被重写的方法是 `default`，那么重写的方法可以是 `default`、`protected` 或者 `public`
	- 如果被重写的方法是 `protected`，那么重写的方法只能是 `protected` 或者 `public`
	- 如果被重写的方法是 `public`， 那么重写的方法就只能是 `public`
5. *重写后的方法不能抛出比父类中更高级别的异常*
6. 可以在子类中通过 `super` 来调用父类中被重写的方法
7. **构造方法不能被重写**
8. 如果一个类继承了抽象类-抽象类中的抽象方法必须在子类中被重写
9. *synchronized 关键字对重写规则没有任何影响*
10. *strictfp-关键字对重写规则没有任何影响*

## 构造方法
```java
public class Wangxiaosan extends Wangsan {
    private int age = 3;
    public Wangxiaosan(int age) {
        this.age = age;
        System.out.println("王小三的年龄：" + this.age);
    }

    public void write() { // 子类覆盖父类方法
        System.out.println("我小三上幼儿园的年龄是：" + this.age);
    }

    public static void main(String[] args) {
        new Wangxiaosan(4);
//      上幼儿园之前
//      我小三上幼儿园的年龄是：0
//      上幼儿园之后
//      王小三的年龄：4
    }
}

class Wangsan {
    Wangsan () {
        System.out.println("上幼儿园之前");
        write();
        System.out.println("上幼儿园之后");
    }
    public void write() {
        System.out.println("老子上幼儿园的年龄是3岁半");
    }
}
```
