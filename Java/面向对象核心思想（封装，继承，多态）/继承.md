# 继承
>子类继承父类，即“子承父业”
>>子类拥有父类除私有的所有成员变量和方法，自己还能在此基础上拓展自己新的属性和方法。
>>主要目的是**复用代码**。
## extends (实现继承)
```java
//class 子类名 extends 父类名 {}
class Child extends Father {}
```
## final (阻止继承)
```java
//final class 类名  {}
final class Father {}
 
// 编译错误：无法继承final类
// class Child extends Father {}
```
## 为什么需要继承
- **复用性**：子类可以直接使用父类的属性和方法，无需重复编写代码
- **扩展性**：子类可以在父类基础上新增功能，或修改原有功能
- **多态性**：继承是多态的前提（后续会关联）
```java
class Animal{
    public String name;
    public int age;
    public int weight;
	// 构造方法
    public Animal(String name, int age, int weight) {
        this.name = name;
        this.age = age;
        this.weight = weight;
    }
    // 这里省略get set方法
    public void sayHello(){
        System.out.println("hello");
    }
    public void eat(){
        System.out.println("I'm eating");
    }
    public void sing(){
        System.out.println("sing");
    }
}

class Dog extends Animal{    // 继承animal
    public Dog(String name, int age, int weight) {
        super(name, age, weight);    // 调用父类构造方法
    }
}
class Cat extends Animal{
    public Cat(String name, int age, int weight) {
        super(name, age, weight);    // 调用父类构造方法
    }
}
class Chicken extends Animal{
    public Chicken(String name, int age, int weight) {
        super(name, age, weight);    // 调用父类构造方法
    }
    // 添加新方法
    public void layEggs(){
        System.out.println("下蛋啦，咯哒咯！咯哒咯！");
    }
}
```
## 重要规则
1. **单继承限制**
	- **一个子类只能有一个直接父类**（Child->Father）
	- 多层继承（Child->Father->Person->Object）
	- 三种**实现多继承**效果的方式，**分别是**内部类、多层继承和**实现接口**
2. **子类构造方法必须调用父类构造方法**
	- 不写 `super()` → 默认调用父类**无参构造**
	- 父类没有无参构造 → **子类必须手动写 `super(参数)`**
	- `super()` 必须放在**子类构造第一行**
3. **访问权限**
	- 子类无法继承（访问）父类的 `private`成员变量 / 方法
	- `protected` 字段和方法可以被其子类，以及子类的子类所访问
4. **不建议**出现与父类**重名成员变量**
	- 如果重名则用`super.fild`区分
## super
>( **指向父类**( 字段 / 方法 / 构造方法) )
- `super.成员变量`：访问父类的成员变量（子类和父类成员**变量重名**时用）
- `super.方法()`：调用父类的方法（子类重写后仍想执行父类逻辑时用）
- `super()`：调用父类的构造方法（必须放在**子类构造方法的第一行**）
```java
class Father {
    protected String surname;    // 子类需要调用该字段修饰符应为protected
    protected String name;
	
    public Father(String surname, String name) {  
        this.surname = surname;  
        this.name = name;  
    }  
	
    public void printName() {  
        System.out.println(this.surname);  
    }  
}

class Child extends Father {  
    private int grade;  
	
    public Child(String surname, String name, int grade) {  
        super(surname, name);    // 调用父类构造方法并初始化
        this.grade = grade;  
    }  
    
    public void printAllinfo() {  
        super.printName();    // 调用父类方法 
        System.out.println(super.name);    // 调用父类成员变量
        System.out.println(this.grade);  
    }
}
```
## 方法重写 @Override
>**子类重新实现父类已有的方法**
### 规则
1. 方法名、参数列表 **必须一样**
2. 返回值类型 **兼容**
3. 访问权限 **不能更严格**
4. 不能重写 `private`/`final` 方法
5. 加上`@Override`注解
---
## 向上(向下)转型
```java
class Father {  
    // 父类定义getInfo方法（为了触发方法重写）  
    public void getInfo() {  
        System.out.println("Father类的getInfo方法");  
    }  
}  
  
class Child extends Father {  
    // 子类重写父类的getInfo方法  
    @Override  
    public void getInfo() {  
        System.out.println("Child类重写的getInfo方法");  
    }  
    // 子类独有的方法  
    public void childOnlyMethod() {  
        System.out.println("Child类独有的方法");  
    }  
}  
  
public class Inheritance {  
    public static void main(String[] args) {  
        // 向上转型：父类引用p指向子类对象  
        Father p = new Child();  
        p.getInfo(); // 输出：Child类重写的getInfo方法  
        // instanceof的使用场景  
        // 判断实际类型是否是Child（true）  
        System.out.println(p instanceof Child);  
        // 判断实际类型是否是Father（true，因为Child是Father的子类）  
        System.out.println(p instanceof Father);  
		// （向下转型）
        // 父类引用调用子类独有方法
        if (p instanceof Child) { // 先判断，避免类型转换异常  
            Child c = (Child) p; // 向下转型  
            c.childOnlyMethod(); // 输出：Child类独有的方法  
        }  
    }  
}
```
### #向上转型
- `Father p = new Child();` （父类引用`p`指向子类对象）
- `p` 的**引用类型**是`Father`，对象**实际类型**是`Child`
- 只能使用父类的方法
	- 方法如果被**重写**`@Override`会执行子类的方法
### #向下转型 
>还原子类特性、调用独有方法
- `Child c = (Child) p;`
- 父类引用`p`指向的**实际对象**必须是该子类
	- `if (p instanceof Child)` 为true
### 对比
|  类型  |           语法形式            |          核心本质          |
| :--: | :-----------------------: | :--------------------: |
| 向上转型 | `Father p = new Child();` | 父类引用指向子类对象（自动转换，无需强制）  |
| 向下转型 |  `Child c = (Child) p;`   | 父类引用转回子类类型（强制转换，需前置判断） |
## #组合
>继承是is关系，组合是has关系
```java
class Book {
    protected String name;
    public String getName() {...}
    public void setName(String name) {...}
}
```
```java
class Student extends Person {
	protected Book book;
    protected int score;
}
```
